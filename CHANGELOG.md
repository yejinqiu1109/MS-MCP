# Changelog

All notable changes are documented here.

## 0.2.0 - 2026-07-19

- Added stateful GUI modeling, Forcite, DMol3, CASTEP, CIF-source, remote-batch, convergence, and Dashboard workflows.
- Added a step-by-step Windows configuration BAT that validates and generates local BAT and Codex TOML files.
- Removed author-machine paths from launchers, Dashboard defaults, GUI loop, examples, and synchronization defaults.
- Added relative-path Dashboard startup/autostart launchers, installation/testing BAT files, package validation, security guidance, and GitHub release manifest.
- Kept arbitrary MaterialsScript, external inputs, and Dashboard writes disabled by default.
- Added upstream attribution and Douyin tutorial information to the public README.
